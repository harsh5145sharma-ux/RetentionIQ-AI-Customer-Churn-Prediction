# RetentionIQ

AI-Powered Customer Retention Intelligence Platform

## Overview

RetentionIQ is an end-to-end machine learning project designed to predict customer churn and identify customers at risk of leaving a service. The project combines advanced analytics, feature engineering, machine learning, and explainable AI to generate actionable retention insights.

## Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Customer Churn Prediction
- XGBoost & Random Forest Models
- SMOTE for Class Imbalance
- SHAP Explainability
- Business Insights & Recommendations

## Tech Stack

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
- XGBoost
- SHAP
